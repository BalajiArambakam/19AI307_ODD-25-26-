# Ex.No:1(E) STRINGS AND MATH FUNCTION

## QUESTION:


## AIM:
To write a Java program to calculate the power of a given number.


## ALGORITHM :
1. Start the program
2. Import java.util.Scanner
3. Create a class named powerx
4. Create the main() method
5. Create a Scanner object to read input
6. Read the base value from the user
7. Read the exponent value from the user
8. Calculate the power using Math.pow(base, expo)
9. Store the result in a variable
10. Display the base, exponent and result
11. Stop the program




## PROGRAM:
 ```
/*
Program to implement a Strings and Math Function using Java
Developed by: Balaji Arambakam
RegisterNumber: 212224230021

 import java.util.Scanner;
public class powerx
{
    public static void main(String[] args)
    {
        Scanner sc = new Scanner(System.in);
        double base = sc.nextDouble();
        double expo = sc.nextDouble();
        double res = Math.pow(base, expo);
        System.out.printf("%.1f raised to the power of %.1f is: %.1f", base, expo, res);
    }
}
*/
```
